    After reaching Vitamin D plateau we need to evaluate response

    September 18, 2023

    Expecting to give 50,000 IU of Vitamin D weekly (or 7,000 IU) daily to many adults,

       along with some Magnesium and Vitamin K2-7

    After the response has plateaued (1 month with loading dose, 3 months otherwise)

    everyone should get a vitamin D test to determin if any change is needed

    || Test results|%|Possible reasons for the result|Possible actions

    <40 ng%%% (<100 nmol/L)|32%|Senior, Autistic, Smoker, Obese, slow responder%%%taking statins, low magnesium, bariatric surgery, %%%Poor health (consuming vitamin D)%%%POOR: liver, genes, gut, pancreas, gall bladder|1) larger dose of Vitamin D, %%%2) gut-friendly vitamin D, %%%3) probiotics%%%4) wait for slow responder

    40 ng-120 ng%%% (100-300 nmol/L)|60%|Right on target|no change needed if feel benefit

    40 ng-120 ng%%% (100-300 nmol/L)|5%|on target, but not feel any benefit%%%Vitamin D probably not getting to cell|Activate Vitamin D receptor%%%    (weekly dosing, Resveratrol, etc.)%%%What to do if Poor Kidney, 3 other genes

    120 ng%%% (>300 nmol/L)|3% |Excessive response%%%Genes, drugs being taken| Reduce Vitamin D dose - perhaps by 50%

    200 ng %%% (500 nmol)|1%|Dangerous response| Should person stop taking Vitamin D%%%Might daily be better for that person?||
